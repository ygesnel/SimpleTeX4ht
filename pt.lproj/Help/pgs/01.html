Sobre o SimpleTeX4ht

SimpleTeX4ht é uma interface gráfica (GUI) Mac OS X para TeX4ht que converte um arquivo LATEX para:

  • HTML ou XHTML
  • MathML
  • Text Encoding Initiative (TEI)
  • DocBook
  • OpenDocument (.odt)

TeX4ht é tipicamente executado a partir da linha de comando em UNIX, algo parecido com: nome do arquivo htlatex "opção1" "opção2" "opção3" "opção4".
SimpleTeX4ht é uma interface gráfica para tornar isso mais amigável para o usuário.
