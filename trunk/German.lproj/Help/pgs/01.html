Über SimpleTeX4ht

SimpleTeX4ht ist eine Mac OS X-Applikation (ein sog. GUI) fürTeX4ht, das LATEX-Dateien in :

  • HTML oder XHTML
  • MathML
  • Text Encoding Initiative (TEI)
  • DocBook
  • OpenDocument (.odt)

SimpleTeX4ht ist ein GUI, eine grafische Benutzerinnenschnittstelle, die das Ganze etwas freundlicher gestaltet.
